Animedia fue un proyecto propuesto por Bictia, el cual suple la necesidad de un cliente que está iniciando un negocio de cinema, el enfoque principal es un portal web desde el cual se pudiera apartar boletería y hacer la compra de esta junto a confitería, además cuenta con tipos de membresía así un usuario puede llegar a ver películas desde este medio, este proyecto fue desarrollado usando Angular, TypeScript, Bootstrap, Node, Express y MongoDB
